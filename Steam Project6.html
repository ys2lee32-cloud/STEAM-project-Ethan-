Rolling time on a wooden ramp

  • Methods: 1. Prepare wooden ramp and balls for experiment 2. We go to somewhere it is not windy and appropriate place to start the experiment(no too much people,no wind) 3. We set the wooden ramp that we created and place it on the ground at a fixed height,place and angle. Surely make sure that it is stable. 4. We set a starting point above the ramp(10cm above) so that every trial goes equal and same 5. We keep all the things such as ramp, starting points height, and the environment the same so that the experiment is fair and accurate. 6. We prepare a stopwatch to measure the time for each balls to roll down and touch the ground 7. We take one specific ball (like metal,plastic,rubber,wood,slime) and put it on the starting point 8. we release the ball softly without pushing it so that it has accurate measurement of time 9. When the ball reaches the ending point/ground, we stop the timer and record the time 10. We repeat the same thing 4 times in each 5 ball and find the average time of each type and texture of the ball 11. We calculate the average time of each type of balls took to touch the ground 12. We compare the average time taken for different texture,type of balls to roll down the ramp and hit the ground
